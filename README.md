## 🚀 Getting Started

Ensure you have the following installed:

- [Node.js](https://nodejs.org/) (LTS version recommended)
- [Angular CLI](https://angular.io/cli)
- [Git](https://git-scm.com/)

### 📥 Clone the Repo

```sh
git clone https://github.com/your-username/my-portfolio.git
cd my-portfolio
```
